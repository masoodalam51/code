Setup Octopress On Windows7

Assalamualaikum,

When you search online for static blog generator in top most position in search engine you got two names, first is jekyll and second is octopress. If you take advice any professional static blogger (Blogger who used static blog generator for generate their blog) is recommend you either jekyll or octopress. If you dont have enough money or powerful web hosting to run Wordpress than Octopress is good for you. Hosting octopress on Github Pages is free of cost and its also take advantage of built in octopress CDN.Its very simple and clean but due to unfamiliar to run a running shell commands and basics of Git,its very difficult to setup a Octopress blog specially on a Windows7 machines.
