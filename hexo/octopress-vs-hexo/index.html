Octopress vs Hexo

Octopress and Hexo both are a static website generator, but platform for both is different. Octopress uses ruby where hexo uses node.js and this is a short comparison of ocropress vs hexo.

Before proceeding further please note this article is based on my experience and i am not ruby nor a node.js programmer. So its looks like i am a general computer user.

This blog used a octopress as a blog generator from 2012-2013, but in the starting of 2014 i accidently found a hexo. You can check this post for detaile about choosing octopress and you can also check this post for migrating from octopress to hexo.

lets start a comparison.

Defination of Both Platform:

What is Octopress?

Octopress is a framework designed by Brandon Mathis for Jekyll, the blog aware static site generator powering Github Pages.

What is hexo?

A fast, simple & powerful blog framework,
powered by Node.js.

Octopress vs Hexo?

1.Speed:

Octopress:

Octopress blog generating speed is very poor, its require a 2-3 minute for generating 24 articles on my windows 7 PC. Its not a octopress fault but its really frustrating.

Hexo:

Hexo generate blog very fast and its justify its own line ‘A fast, simple & powerful blog framework’, as compare to octopress its generates 24 article in seconds. Shocking Octopress Fans, but its true.

Winner:

Hexo

2.Complexity:

Octopress:

For non technical (Programmer) user octopress is a nightmare. I remembered when i was thinking to create a blog using octopress in august 2012, i unable to found any article which is guide me to simply start a blog. After serching and visiting lots of web-pages i create my own solution which is also shared as a tutorial Setup Octopress On Windows7.

Hexo:

For non technical user (Programmer) hexo is very easy, Install only one software node.js and start your own blog. Its very simple and easy.

Winner:

Hexo

3.Software Update:

Octopress:

Official version of octopress is not updated from July 2011.

Hexo:

Hexo updated in at least 2-3 month. Its Actively maintained.

Winner:

Hexo

4.SEO:

Octopress:

Out of the box version of octopress is not a seo friendly, but lots of customization is available.

Hexo:

Default version of hexo is seo friendly. No customization is needed.

Winner:

Hexo

5.Themes and Plugin:

Octopress:

Lot of themes and plugin available for octopress because octopress is a jekyll for blogs so using some customization nearly all themes and plugin which is made for Jekyll is available for octopress.

Hexo:

Reasonable amount of theme and plugin available for hexo but not as huge as octopress.

Winner:

Octopress

6.Customization:

Octopress:

Octopress customization is bit difficult as compare to hexo.

Hexo:

Customization of hexo is simple as compare to octopress.

Winner:

Hexo

7.Support:

Octopress:

Octopress have a great community and its support mainly in English.

Hexo:

Hexo also have a great community but one main difficulty for English user is many discussion happen in Chinese.

Winner:

Octopress

8.Age:

Octopress :

Octopress built upon a ruby and Jekyll and both are older than node.js platform. So its have a proven track record and you found almost all answer about your question.

Hexo:

Node.js is a new platform as compare to ruby so hexo is also newer than octopress.

Winner:

Octopress

Article is based on my personal experince and i am not a programmer or expert on ruby or node.js.
