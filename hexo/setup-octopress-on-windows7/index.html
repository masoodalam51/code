Setup Octopress On Windows7

Assalamualaikum,

When you search online for static blog generator in top most position in search engine you got two names, first is jekyll and second is octopress. If you take advice any professional static blogger (Blogger who used static blog generator for generate their blog) is recommend you either jekyll or octopress. If you dont have enough money or powerful web hosting to run Wordpress than Octopress is good for you. Hosting octopress on Github Pages is free of cost and its also take advantage of built in octopress CDN.Its very simple and clean but due to unfamiliar to run a running shell commands and basics of Git,its very difficult to setup a Octopress blog specially on a Windows7 machines.

Advantage and Disadvantage Of Octopress

Using Octopress get you some advantage like your posts are always safe in your local machine and due to no server side operation its not slow down your website because no extra queries or function perform in background. But you miss some best plugin which is available on wordpress like third party commenting system is not required, automatic search engine optimization using various plugin, beautiful contact us page with many templates and one major disadvantage according to you cant create a multi-author blog which is necessary if you encourage guest posting or want to maximize your blog audience.

Note: Now i am using hexo as my blog generator. Read this comparison between Octopress VS Hexo. Also Check this post where i explain why i choose Hexo Over Octopress

This is quickest and easiest guide to setting up Octopress on a Windows7 machine for publishing your blog with GitHub pages.

3) Install Development Kit.In this tutorial i use DevKit-tdm-32-4.5.2-20111229-1559-sfx.exe and extract files into C:/RubyDevKit.
