3 Software for Design Circuit Diagram

As a electronics engineer designing a circuit diagram is not a big deal. when i was a new student in electronics field teachers teach us to design a circuit diagram using hand, Its a fun game and i really enjoyed this. But when complexity was increased drawing a circuit diagram using hand is really a bad idea because i want a cleaner tracks and want a better circuit diagram using small piece of PCB (Printed Circuit Board). Now that time teachers teach us how to design circuit diagram using software. When you entered into a software world for designing a circuit diagram you will be amazed. I was really amazed like how space saved while designing circuit diagram and simulation of circuit is very interesting. In this post i suggest you a 3 software for design circuit diagram.

This three software which is listed below is for beginners, that’s means this post is not for professional or advance level user.

This is a first post in TECHELEX Electronics Software Series.

Software for Circuit Diagram Design

1. FidoCadj

a free graphical editor for (more than) electronics,FidoCadJ is a simple vector graphic editor. You can draw whatever you want, and FidoCadJ comes with a large library of symbols related to the electrical and electronic engineering.

FidoCadJ is a free user-friendly vector graphic editor for MacOSX, Linux, Windows and Android:

  • A simple schematic capture with a built-in set of electrical symbols and footprints;
  • A versatile graphical editor with user-defined libraries;
  • Easy-sharing text format for web forums and blogs;
  • Export in several graphical formats: PDF, EPS, SVG, PNG, JPG, and PGF/TikZ for LaTeX;
  • User interface in English, French, Italian, German, Chinese, Spanish, Czech, Japanese and Dutch.

FidoCadj is really simple and light weight program and good news is its in active development.

Download From Here: FidoCadj

2. Circuit Maker 6 Pro

CircuitMaker is the most powerful, easy-to-use schematic capture and simulation tool in its class. Using CircuitMaker’s advanced schematic capabilities, you can design electronic circuits and output netlists for TraxMaker and other PCB design tools and autorouters. You can also perform fast, accurate simulations of digital, analog and mixed analog/digital circuits using CircuitMaker’s Berkeley SPICE3f5/XSpice-based simulator.

Note: As you seen in picture you found a simulation of Collector Coupled Astable Multivibrator. This software containing simulation facility and as a beginner it was my personnel favorite.

Download From Here: Circuit Maker 6 Pro

3. TinyCAD

TinyCAD is a program for drawing electrical circuit diagrams commonly known as schematic drawings. It supports standard and custom symbol libraries. It supports PCB layout programs with several netlist formats and can also produce SPICE simulation netlists. It is also often used to draw one-line diagrams, block diagrams, and presentation drawings.

  • Flat or Hierarchical Schematic Entry
  • Many popular PCB Layout compatible netlist formats
  • Convenient support for embedding graphical images in drawings

Download From Here: TinyCAD
