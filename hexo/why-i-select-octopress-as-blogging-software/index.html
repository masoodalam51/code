Why I Select Octopress As Blogging Software

Assalamualaikum,

For a neat and lightweight blogging,I tried many blogging software but no one full fill my need.i always think to write a blog using static file and because static software is always faster than dynamic bloging software.

Note: Now i am using hexo as my blog generator. Read this comparison between Octopress VS Hexo. Also Check this post where i explain why i choose Hexo Over Octopress

After saw a Forum software which use a flat-file as a database myUpb.i continuously thinking about a static blogging software,after a googling i found a wordpress like blog which uses a flat-file as a database and also a free FlatPress.but in FlatPress comment plugin like a insertion of disqus is not easy and not configure in right way.after Flatpress i tried a Jekyll-Bootstrap as a blogging Software but i unable to understand it and unable to configure it[Because i am unfamiliar to Github].After a Jekyll-Bootstrap i tried a [[Octopress]] and i love it,its a simple in use but due to unfamiliar environment to ruby i learn to publish a blog using a Octopress + Windows7 in 15 days using a tutorial available in internet.
