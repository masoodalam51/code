Why I Switch My Blog From Octopress To Hexo

Assalamualaikum,

When i start my first personal blog i am totally unaware about how to create a blog and how to maintain it. But internet and Google is always your best friend for making any thing in Awesome manner. I use Blogspot first, its a simple and easy platform but i want something extra so decide to use wordpress. Wordpress is world largest platform for hosting your blog or you install it on your server, but its also include a many feature which is not require in my blog so i think to switch to new blogging engine which is Octopress. But using it i realize its generation speed is very slow. Now my hunt is on for faster and Octopress like generation software i found a New And Better alternative of Octopress Which is called Hexo.

Why I Switch My Blog From Octopress to Hexo

Octopress is very good platform and its tag line is attract me to start my blog using Octopress.

A blogging framework for hackers.

Its very user friendly but when you are generating a website its takes too much time, i have only 24 articles but its takes 2-3 minutes for generating entire website. so i decide to switch my blog to any other blogging software which is similar to Octopress and my main criteria for choosing any blogging software is speed of generation. After lot of googling i found a Hexo which is similar to Octopress but its takes only 2-3 second which is very fast as compared to Octopress. When switching any platform for website its takes lot of headache and hard work for converting website or blog from one software markup to another, but in my case nothing to change in my article and code simply copy previous articles markdown files to Hexo directory. Running generation command and my blog works again.
