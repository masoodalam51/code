Hospital Staff Tracking System

Have you ever wondered how nice it will be if you would be able to locate a person without using GPS? The said project does the same. This project is very helpful in closed premises such as hospitals schools etc. You are searching for a particular person nearest to you every person of that institute will be provided active RFID cards which more or less works as a GPS. The person nearest to you will be automatically tracked with the help of RFID provided to him and you can easily locate the person at the centralized server.
This is very useful in hospital where you frequently require hospital staff to attend the patient. With the implementation of this system it becomes easier to search for the nearest nurse, ward boy or a doctor. This in turn can be very crucial as precious time can be saved and attend an emergency faster. This can helpful to lot of patient.
Every RFID will be provided with an exclusive code that will be identified uniquely on the centralized monitoring system. The current location of each member of hospitals will be updated on the centralized monitoring system making the tracking of the person very easy using hospital staff tracking system.
